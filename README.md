# 🌐 Estudos de Redes - Cisco Packet Tracer

Bem-vindo ao meu repositório de laboratórios práticos de Redes de Computadores. Aqui documento minhas simulações utilizando o Cisco Packet Tracer, explorando desde a comunicação básica na Camada 2 até roteamento avançado, IPv6 e análise de desempenho.

## 📂 Laboratórios Disponíveis

* **[Unidade 1: Hardware e Cabeamento](./U1-Hardware-e-Cabeamento)**: Comunicação básica entre hosts, configuração de switch e tabelas MAC.
* **[Unidade 2: Arquitetura TCP/IP](./U2-Arquitetura-TCPIP)**: Interligação de sub-redes distintas utilizando roteadores e configuração de Gateways.
* **[Unidade 3: Protocolo IPv6](./U3-Protocolo-IPv6)**: Transição para IPv6, roteamento unicast e configuração de prefixos /64.
* **[Unidade 4: Gerência de Desempenho](./U4-Gerencia-de-Desempenho)**: Análise de tráfego, throughput e monitoramento de saturação de rede.
* **[Unidade 5: Segmentação de Redes](./U5-Segmentacao-de-Redes-VLANs)**: VLANs, Trunking e Roteamento Inter-VLAN (Router-on-a-Stick).

*Laboratórios desenvolvidos como parte da grade curricular e aprofundamento prático em infraestrutura de TI.*
